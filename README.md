# Web Dev Practice

This repository contains my backend and MongoDB practice code while learning web development.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Project Structure
- `index.js` → Main server file
- `books.js` → Database/model related code
- `package.json` → Project dependencies

## Purpose
I created this repository to:
- Practice backend development
- Learn MongoDB and Mongoose
- Build consistency by pushing code daily
- Improve my full-stack development skills

## How to Run

1. Install dependencies
```bash
npm install
```

2. Start MongoDB server

3. Run the project
```bash
node index.js
```

This repository is part of my journey from beginner to advanced web development.
